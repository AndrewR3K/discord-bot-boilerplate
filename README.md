
# Node.js Discord Bot Boilerplate

> Well, a Node.js Discord bot boilerplate... Beep B00p

## Installation

```bash
$ yarn install
```

## Setup
You will need to setup an environment file with the following:

```
TOKEN=<Your Discord token>
NODE_ENV=development
```

## Build - Development

```bash
$ yarn dev
```

## Build - Production

```bash
$ yarn start
```
